
#### プロセス周り
稼働中のプロセスをリアルタイム表示 : `top`

プロセス確認 : `ps -ef | grep <ps_name>`

プロセスを止める(通常) : `kill -9 <ps_num>`

プロセスを止める(強制) : `kill -KILL <ps_num>`

#### サービス周り
起動/停止/再起動 : `servie <service_name> start|stop|restart `

自動起動登録/解除 : `chkconfig <service_name> on | off`
